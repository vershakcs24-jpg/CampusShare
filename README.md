# 🎓 CampusShare

A full-stack campus marketplace built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. CampusShare enables college students to securely buy, sell, and exchange products within their campus through a modern, responsive, and user-friendly platform.

---

## Overview

This project demonstrates the implementation of core full-stack web development concepts, including:

* MERN Stack Architecture
* JWT Authentication & Authorization
* RESTful API Development
* MongoDB Database Integration
* Cloudinary Image Uploads
* Real-time Buyer-Seller Chat
* CRUD Operations
* Responsive UI Design
* Secure Backend Development

The objective of this project was to build a scalable marketplace platform that simplifies campus buying and selling while strengthening full-stack development skills.

---

## Features

* Secure User Registration & Login
* JWT-based Authentication
* Create, Edit & Delete Product Listings
* Upload Multiple Product Images
* Cloudinary Image Storage
* Buyer-Seller Real-time Chat
* Search Products
* Category-based Browsing
* User Profile Management
* Responsive Design for All Devices
* Modern and Interactive User Interface

---

## Application Flow

User Registration/Login
↓
JWT Authentication
↓
Browse Marketplace
↓
Search or Filter Products
↓
View Product Details
↓
Chat with Seller
↓
Buy / Sell / Exchange Products
↓
Manage Profile & Listings

CampusShare provides a seamless experience for students to discover products, connect with sellers, and manage their marketplace activities securely.
